# Aetheris Ring — Product Landing Page

A premium, modern, and interactive single-page landing site for the **Aetheris Ring**, a futuristic health wearable crafted from titanium with quantum biosensors.

## 🚀 Live Demo and Screenshots
Below are the key sections of the product landing page:
- **Hero Area**: Stately dark mode presentation with animated glowing rings and floating product visuals.
- **Stats Banner**: Key metrics including 7-day battery life, 50m water resistance, and weight.
- **Biometric Technology Grid**: Responsive two-column breakdown detailing sensor alignment and companion app dashboard.
- **Interactive Configuration Tool**: Live finish selection (Titanium, Stealth Black, Aurora Gold) and real-time finger circumference slider tool to determine user ring sizing.
- **Pre-Order Gateway**: Validation-enabled HTML form with instantaneous mock queue registration feedback.

---

## 🛠️ Technology Stack
- **HTML5**: Semantic tags for clean structure and strong SEO indexing.
- **Vanilla CSS3**: Tailored layout variables, glassmorphic panels, CSS keyframes, and custom transitions. 
- **Vanilla Javascript**: Interactive selectors, live sizing interpolation logic, and feedback management.

## 💻 Getting Started

To launch and explore the application locally:

### Option 1: File Launch
Double-click `index.html` to open the file directly in any modern web browser.

### Option 2: Local HTTP Server
Run a local Python static server from the project directory:
```powershell
python -m http.server 8080
```
Then, open your browser and navigate to:
`http://localhost:8080`
